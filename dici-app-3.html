🔥 Connexion Firebase (synchronisation)
Visible dans Realtime Database → onglet Données, en haut de la page.
Ces infos sont dans la console Firebase → Paramètres du projet → Tes applications → Config SDK.
📋 Guide de création Firebase (5 min)
1
Va sur console.firebase.google.com et connecte-toi avec ton compte Google.
2
Clique "Créer un projet", donne-lui un nom (ex : dici-rayon), désactive Google Analytics si demandé.
3
Dans le projet, clique Realtime Database → "Créer une base de données" → Choisir Démarrer en mode test.
4
Reviens à l'accueil, clique l'icône </> pour ajouter une app Web. Donne-lui un nom, puis copie les valeurs apiKey, projectId et appId dans les champs ci-dessus.
5
Envoie ce même fichier HTML à ta collègue, elle renseigne les mêmes valeurs Firebase + son prénom. C'est tout !
